# Installation

 1- make sure to ensure meta_libs after qb-core directly, it will show like this: 
 ensure qb-core 
 ensure meta_libs 
 ensure [qb] 
 ensure [standalone] 
 ensure [voice] 
 ensure [defaultmaps]
 2- insert the sql 
 3- add new items for ingrediants or just change them
